# Week 8 - focus on user privilege and database security

## Key areas
* User privileges - Admin, Customer, Warehouse, Analytics
    - Least privilege & Seperation of Duties

* Security and data integrity
    - SQL-Injections - Prepared Statements, Stored Procedures
    - Prevent mistakes - limit write access and version control & logging
    - Protect admin credentials - secrets, .env folder, vault

* GDPR and sensitive data
    - Split data into types: Customer data, Business data, Sensitive customer data
    - Encryption / Data seperation
